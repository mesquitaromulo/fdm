Problemas Comuns GIT

Como remover arquivos da área de stage

git reset

Desfazendo modificações não salvas

git reset

git clean -df

git checkout -- .

Como desfazer o último commit

Desfazer o último coomit sem desfazer as modificações nos arquivos:

git reset --soft HEAD~1

Deletando commits e também modificações nos arquivos

Voltar o projeto ao estado de um dado commit(deletar commits posteriores a esse commit):

git reset --hard (código do commit)

Voltar o projeto ao estado do penúltimo commit:

git reset --hard HEAD~1

Atualizar o repositório local em relação ao remoto

git pull (nome do remote) (nome da branch)

Como resolver push rejeitado

Precisa atualizar o repositório local

git pull (nome do remote) (nome da branch)

Como sobrescrever um histórico no Github

git push -f (nome do remote) (nome do branch)
